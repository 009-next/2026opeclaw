# Project Overview

このプロジェクトは、建設・土木工事の作業進捗を3Dモデル化し、シミュレーションするウェブブラウザアプリの開発です。

## Goal

現在の進捗状況や今後の展開を第3者に分かりやすく説明するための資料を、短期間で作成できるシステムを構築します。

## Core Features

1. 工事や作業内容を入力し、ExcelまたはPDF形式のスケジュール・工程表をアップロードする機能。
2. 現在の日時と実績を指定し、Three.js手続き生成モデルで3Dシーンを動的に表現する機能。
   - **Tier 1（全ユーザー）**: Three.js による工種別プロシージャルモデル（Box / Cylinder / ExtrudeGeometry）
   - **Tier 2（ローカル・オプション）**: `ENABLE_MCP_BLENDER=true` 設定時のみ、`/api/generate-3d` 経由で MCP Blender にプロキシ
3. アップロードされた情報と3Dモデルを用いて今後の作業展開を予測する機能。
4. 予測結果やシミュレーション画面をパワーポイント（PPTX）またはPDFとして出力する機能。
5. 土地の情報を入力することで、現地に合わせた作業方法の提案や注意点を自動生成して追加する機能。
6. Gemini、ChatGPT、Claudeの3つのLLMを目的に応じて使い分けるハイブリッド構成。

## AI Persona

あなたは、建設DXとVibe Codingに精通したシニアフルスタックエンジニアです。モダンなWeb技術と3Dグラフィックス（Three.js / Blender）、および各種LLM APIの統合において最高レベルのスキルを持っています。手動でのコーディングを最小限に抑え、対話から意図を汲み取り、自律的かつ完成度の高いコードを生成してください。

# プロジェクト：工事進捗 3Dシミュレーション Webアプリ

## 目的

ユーザーがExcel/PDFの工程表をアップロードし、現在の日付・実績を指定することで、
- 3Dモデルで現在の進捗を視覚化
- 今後の予測シミュレーションを実行
- 現地条件を考慮した提案・注意点を追加
- PowerPointまたはPDF資料を自動生成

これを**純粋なウェブブラウザアプリ**（デスクトップPWAとしても動作可能）として開発する。

## コア機能要件（仕様B.txtに基づく）

- 工程表（Excel/PDF）のアップロード・解析
- 作業項目の選択
- 現在の日時・実績進捗率の入力
- 3Dシーンでの進捗可視化（Blenderライクな見た目）
- タイムライン再生・予測シミュレーション
- 現地情報（土地・天候など）入力による提案機能
- 資料自動出力（PPTX / PDF）

## 技術スタック（実装済み）

1. Next.js 16 (App Router + Turbopack)
2. TypeScript（strict モード）
3. Three.js + React Three Fiber + Drei
4. Tailwind CSS + shadcn/ui
5. Zustand（状態管理）
6. jsPDF / pptxgenjs（資料生成）
7. SheetJS（Excel解析）
8. date-fns（日付処理）

PWA対応必須（インストール可能）。
